Actions

Trigger updates manually without using curl or other command-line tools.

Partial Update

Update all files modified between two commits. The 'till' field defaults to HEAD if not specified.

Note: If ADMIN_PASSWORD is not set on the server, authentication is not required (leave password blank).

Full Update

Run the full update script to process all files in the repository.

Note: If ADMIN_PASSWORD is not set on the server, authentication is not required (leave password blank).
